# DevAir

Flight reservation / onward-ticket product (Next.js).

## Stack
- Next.js (App Router)
- Local MVP: email auth stub; Postgres/Stripe in later waves

## Develop

```bash
npm install
npm run dev
```

Smoke:

```bash
npm run test:mvp
```

## Repo scope
This GitHub repo is **product code only**. Platform OS (Paperclip, Ruflo, vault, swarms) lives in Agents-Dev — do not commit harness/agent runtime files here.
